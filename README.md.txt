Requirements for Matlab
- install Matlab Waivelet Toolbox

Instructions:

PlotElectrodeSignal.m: Shows SWRs + raw signal for the given recording on a single recording
path - set the top directory with data, for the given day
channelList - set to the number with the channel to display

PlotMultipleElectrodeSignal.m: Shows raw signal on all of the electrodes
same params as for PlotElectrodeSignal.m

ymaze_trial_trace.m: Shows data raw data on the channel along with the trajectory of the mouse on the ymaze, last subplot shows spectrogram of the signal
relies on the tracking data being available
